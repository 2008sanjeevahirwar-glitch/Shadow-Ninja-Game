**Requirements:**
```bash

```

**Run:**
```bash

```

**Features:**
- Full graphical window using Pygame

- Sprite-based rendering
- FPS: 60 frames per second
- Cross-platform (Windows, Mac, Linux)

- Easiest to understand for beginners
- Great for rapid prototyping
- Excellent graphics capabilities
- Rich game development libraries

**Cons:**
- Requires Python installation
- Pygame dependency needed

## üéÆ Gameplay

All versions feature identical gameplay:

**Controls:**
- **Arrow Keys / A-D:** Move left/right
- **Space:** Jump (double jump available)
- **S:** Stealth mode
- **F:** Attack enemies
- **Shift:** Dash (when unlocked)

**Objective:**
- Collect coins (2 coins visible)
- Avoid enemies and traps
- Survive with remaining health
- Collect coins to unlock dash ability
